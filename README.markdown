# MXIT PC Client

## Building

Requires >=cmake-2.6.0
Requires Qt development libraries

  % cd /path/to/mxit
  % cd build/
  % cmake ..

At this point, fix any errors reported by cmake (missing libraries..)
To build in debug, use `cmake -DCMAKE_BUILD_TYPE="Debug" ..`

  % make
  % make install
